# LIFE-EXPECTANCY-PREDICTION

Exploratory Analysis on factors influencing Life Expectancy to understand what area(s) should be given importance in order to efficiently improve the life expectancy of the population?
The steps performed in R include:

1. Statical Summary Analysis

2. Data Cleaning and Preprocessing
  2.1 Remove Missing Values
  2.2 Data Imputation
  2.3 Outlier Removal

3. Data Visualization using histograms, boxplots, scatter plots, heatmaps, bargraphs, pieplots, linegraphs using ggplot library. 

4. Correlation Analysis using Pearson and Spearman Correlation.

5.  Regression model for predicting life expectancy
